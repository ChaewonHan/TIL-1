# 자바 직렬화 (Java Serialize)
- Java 내부 시스템(JVM의 메모리에 올려진)에서 사용되는 객체나 데이터를 외부에서 사용할 수 있도록 byte 형태로 변환하는 것. (다시 객체로 역직렬화하는 것을 포함)
- `Java.io.Serializable` 인터페이스를 상속받은 객체와 Primitive 타입의 데이터가 직렬화의 대상이 될 수 있다.
